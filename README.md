# TastyIgniter Docker

clone the repo

    git clone https://github.com/NTUEEInfoDep/NTUEEWEEKPOS.git

Required to prepare .env file for installation!

Run with docker compose for automatic database configuration

    cd NTUEEWEEKPOS
    docker-compose up -d
